In this folder you will find two zip files both used to create graphs to visualize data collected. We recommend that you first use the ParkingDataCommuterWithPlotly first unless you
have issues with uploading/loading a csv file. To run the code simply run the first cell if it is a sql file and skip the second cell. If it is a csv you are uploading skip the first cell and run the second making sure the name of the file to be scanned is in the code correctly. If you would like to change the zones you wish to look at make sure you confirm the zone id with the chart provided in the final lab report.
